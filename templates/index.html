{% if key == 'lasso_pca' %}
Why Use This Model
🥇 This is the #1 ranked model — use it as your primary prediction.
L1 regularisation automatically zeroes out weak PCA components, keeping only the most influential signals.
Highest CV R²=0.969 — best generalisation on unseen data.
Slightly more aggressive than Ridge — good when market has clear momentum.
Critical Input Rules
Gold_Price is the most important input — enter today's exact spot price.
Return_7d / Return_14d / Return_30d must be decimal fractions: 1.6% gain = 0.016, NOT 1.6.
Gold_Silver_Ratio = Gold_Price ÷ Silver price (e.g. 3320 ÷ 32.8 = 101.2).
Gold_VIX_interaction = Gold_Price × current VIX index value.
Volatility_7d / Volatility_30d = std of daily RETURNS as decimal (0.008 = 0.8%/day).
